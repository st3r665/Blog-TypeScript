# Simple Blog 后端

## 项目简介
Simple Blog 是一个简单的博客后端服务，提供文章的增删改查等基础功能。

## 主要功能
- 用户注册与登录
- 文章的发布、编辑、删除
- 文章列表与详情查询
- 评论功能

## 如何运行
1. 安装依赖：`npm install`
2. 启动服务：`npm start`

## 依赖
- Node.js
- Express
- 其他依赖请查看 package.json
